# Voyage Explorer

Projet web simple permettant de rechercher des vols et des hôtels.
Le site sert principalement de projet d'apprentissage pour pratiquer **HTML, CSS et Vue.js**.

## Structure des pages

### index.html

Page d'accueil du site.

Contenu principal :

* Présentation de la plateforme
* Explication de l'objectif du site
* Liste des avantages :

  * comparer les vols
  * trouver des hôtels
  * préparer un voyage facilement

### vols.html

Page de recherche de vols.

Fonctionnalités :

* Formulaire de recherche :

  * ville de départ
  * ville d'arrivée
  * date de départ
  * date de retour
* Suggestions automatiques de villes
* Affichage dynamique d'une liste de vols disponibles
* Chaque vol peut être sélectionné pour voir plus de détails.

### infoVol.html (page détail du vol)

Page affichant les informations détaillées d'un vol sélectionné.

Contenu :

* ville de départ
* ville d'arrivée
* date du vol
* prix
* informations supplémentaires sur le vol

### hotels.html

Page de recherche d'hôtels.

Fonctionnalités :

* champ de recherche de ville
* suggestions automatiques de villes
* bouton pour lancer une recherche d'hôtels

### contact.html

Page permettant aux utilisateurs de contacter les créateurs du site.

Contenu :

* formulaire simple :

  * nom
  * email
  * message
* bouton d'envoi

### apropos.html

Page expliquant le projet.

Contenu :

* description du projet **Voyage Explorer**
* objectif pédagogique
* technologies utilisées

## Navigation

Toutes les pages possèdent une barre de navigation commune permettant d'accéder aux sections suivantes :

Accueil
Vols
Hôtels
À propos
Contact

* l'utilisation de Vue.js pour les interactions dynamiques
* l'organisation d'un petit projet web multi-pages
